作为一个普通的消费者来说，很有可能在不久的未来入手MAC，所以把自己在windows上喜欢用的习惯用的软件都做一个记录，留作纪念。
我的笔记本：联想的K29，2013年入手，有五年的windows使用经历，前一部笔记本是东芝L600。
我的桌面

很多图标都经过修改。

聊天软件：

  • QQ：真心离不开，即使是这个移动互联网盛行的时代，依旧是主力。可以使用轻聊版，有比肩MAC版本的清爽，也留下了完整的体验。

  • 微信：偶尔在电脑上用用，必须用手机才可以登录。虽然很麻烦，但是在电脑上打字聊天确实比较方便，不知道大家朋友里有没有人只用微信的？反正我有，那些可恶的手机党啊！

  • 飞信：部分代替短信的工具，虽然上的越来越少了，但是在上海还是有很多人用的，特别是企业里，加上好友后发通知都是免费的。

  • 阿里旺旺：也是离不开啊，因为购物，如果发生纠纷，那么聊天记录是最好的保证。

云同步软件：

  • 百度云管家，使用体验很一般，真的，经常出现问题，特别是笔记本睡眠醒来后，不过功能实在太过于强大了，其实我对于云的概念就停留在备份和在线操作。都来自于苹果的Time Machine（虽然我没有用过），多时间版本的文件备份和实时同步，超大的容量以及局域网的互联互通（曾经看到过有人用最新的Time Machine用蓝光视频）。那么本地可以，云是否也可以呢？百度云几乎可以做到这点。

清理电脑软件：

  • CCleaner：不喜欢360或者其他的类似的软件，主要是功能太多，我不需要它提醒我开机多少时间，不需要知道我有多少漏洞没补，就是简单的清理我的电脑注册表垃圾，不过我并不认为缓存是垃圾。

杀毒软件：

  • MSE，我也觉得一般，不过它几乎没有打扰过我，所以我安装了它。支持在任务栏中显示搜索中-完成-未保护三个状态（这是我最喜欢的，免得有cpu大量被占有也不知道是什么软件的问题）

音乐软件：

  • iTunes：本来也是习惯QQ音乐的，但是买了iPhone后就觉得没有选择了，也习惯了iTunes，觉得这个界面特别漂亮。

不过需要大量的时间管理维护这些专辑封面哦。

视频软件：

  • Potplayer：原来我也是QQ影音的忠实用户，不过QQ影音因为腾讯很久没有更新了，在很多细节上确实没有Potplayer好用，不过Potplayer最好也是选择32位的，64位在解码方面没有32位的来的强悍，我的电脑在QQ影音和64位的Potplayer中播放4K视频都卡，但是在32位中就比较流畅了。附上我的皮肤图（有需要的留言哦！）：

Connectify 3.7

  • 在我的K29上很稳定，不过只能连接8个wifi设备，目前从来没有因为网络限制出现移动设备不能上网的问题，只要电脑可以上网，手机等移动设备一定可以上网，不过我不推荐使用，一般的用户还是使用猎豹wifi比较好，因为做的漂亮，功能也强大些，就是更新比较频繁。

Preme for windows

  • 真是牛人的软件，几乎让鼠标在windows上的体验比肩mac上触摸板上的体验，如果说windows的笔记本的触摸板是渣，那么因为这软件，mac的鼠标同样是渣。上图（只有英文版，不过有视频演示）：

USB Extension

  • 其实就是一个usb设备显示在桌面的插件，很多人习惯360的那种方式也是可以的，我是喜欢MAC的那种方式，讨厌浮窗的。上图：

ASUS AI Charger

  • 可以让iPad在windows上充电的软件。

Ditto

  • 我觉得很不漂亮，但很实用，剪切板管理软件。

Listary

  • 非常非常强大的本地搜索软件，只是比Mac的差了很多，不支持网上搜索，也不支持编辑搜索的软件。比Mac那款神器的搜索软件差不少，但是胜在一个快，本地搜索几乎是瞬间的。也支持everything（虽然我不用everything这软件，但是很多大神都说它是神器）

Secret folder

  • 加密文件夹，可以让文件夹在文件管理器中凭空消失，不过不知道拆下硬盘后是否可以加密。

Media Preview

  • 几乎支持所有的视频预览图，直接上图看效果：

CBXManager

  • 在百度贴吧里发现的软件，可以实现一个我很喜欢的功能，上图：

epub电子书的封面可以显示了！

软媒时间

  • 可以让电脑的时间显示比手机的更棒。上图：

下载软件：

  • QQ旋风：其实不好用，腾讯对它的支持也少了，在扫黄后，还是可以用来下载些违法的内容的。

  • 百度云管家：扫黄前神器，扫黄后只能下些普通的电影电视剧了，而且有些电影也会报错。

  • 维棠下载：目前也是下载的一款神器，对于在线视频的分析下载是没有软件可以比拟的（不过有广告，去掉弹窗，一切都可以忍，据说可以下载付费电影，没实验过，追剧很方便）

桌面应用管理的软件：

  • 酷鱼快速启动：可以用来管理所有的应用软件，上图：

软媒桌面

  • 类似MAC的DOCK栏效果，上图：

两者共同解决了windows应用点击后无明显反应的问题！

浏览器软件：

  • 360浏览器：o(︶︿︶)o 唉，没办法，习惯了，功能目前来说已经不算强大了，但是我认识的人都在用这个浏览器，也确实有无法代替的功能，比如右键关闭标签，比如皮肤，比如显示的效果，其实说真的就是习惯了，现在像UC浏览器其实在功能上都比360要强大了，但是就是不想用。

  • AdSafe：虽然用久了闪退，但是确实是一个屏蔽广告最好的软件，支持浏览器flash广告，也支持视频网站的视频广告，更加支持软件自带的弹窗广告，而且一键操作！

  • 办公软件：还是选择了WPS，虽然微软的OFFICE是主流，但是还是觉得WPS来的可靠些，不希望在关键时候出来一个激活信息。

完结。
